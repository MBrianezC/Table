How to create a table in HTML and manage them correctly

Creating a table in html can be challenging for begginers, but understanding its structure makes the process much easier!

Structure

Since a table is an HTML tag, it is important to first understand how a basic html structure works. Learn more about it here

Once you learned how HTML tag works, you are 70% ready to make your tables.

Understanding

First of all, create in a tag inside the <body> called <table>.

In the HTML structure, you need a tag that defines what will happen.
When you type <table> nothing appears on the website yet, but the code is already set up to begin the magic.


<thead>

Like in HTML, it sets the head. 'T' stands for table.

<tbody>

Like in HTML, it sets the body. 'T' stands for table.

<tfoot>

Like in HTML, it sets the footer. 'T' stands for table.



You might be wondering, what about the tags for adding content?

If you have already learned about <ul> and <ol> you know they define lists, while <li> element contains the actual list items.

Similarly, for table, we use <tr> to define rows inside the code and <td> to show they content in the web browser.

Almost there
